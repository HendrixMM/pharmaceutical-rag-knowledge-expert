## Security Issues

**üîí NEVER report security vulnerabilities as public issues.**

### How to Report Security Vulnerabilities

1. **Private Reporting (Preferred)**:
   - Use [GitHub Security Advisories](https://github.com/hendrixmm/RAG-Template-for-NVIDIA-nemoretriever/security/advisories/new)
   - Fill out the security advisory form
   - Provide detailed information

2. **Include in Report**:
   - Description of vulnerability
   - Steps to reproduce
   - Potential impact
   - Suggested fix (if available)
   - Your contact information

3. **Response Timeline**:
   - **Initial Response**: Within 48 hours
   - **Assessment**: Within 1 week
   - **Fix & Disclosure**: Coordinated disclosure timeline

See [Security Policy](SECURITY.md) for complete security reporting procedures.

## Frequently Asked Questions

### General

**Q: Is this project free to use?**
A: Yes, the project is open source. See [LICENSE](../LICENSE) for details. NVIDIA API usage follows their pricing model.

**Q: What's included in the free tier?**
A: See [FREE_TIER_MAXIMIZATION.md](FREE_TIER_MAXIMIZATION.md) for free tier limits and optimization strategies.

**Q: Can I use this for commercial purposes?**
A: Check the [LICENSE](../LICENSE) file and NVIDIA API terms of service.

### Technical

**Q: What Python version is required?**
A: Python 3.10 or higher. See [Installation Guide](INSTALLATION.md).

**Q: Do I need a GPU?**
A: No, the system uses cloud APIs by default. GPU is optional for local acceleration.

**Q: Can I use my own models?**
A: Yes, see [Configuration Guide](CONFIGURATION.md) for custom model configuration.

### Support

**Q: How do I get help with an error?**
A: Check [Troubleshooting Guide](TROUBLESHOOTING_GUIDE.md), search issues, or create a new issue.

**Q: How long until my question is answered?**
A: See [Response Times](#response-times) section above.

**Q: Can I request features?**
A: Yes! Use the feature request template in [GitHub Issues](https://github.com/hendrixmm/RAG-Template-for-NVIDIA-nemoretriever/issues/new/choose).
